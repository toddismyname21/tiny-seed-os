# FIELD LEAD QUICK START GUIDE
## Tiny Seed Farm - Supervisor Guide

---

## Your Role

You supervise field operations, coordinate crew, and track daily progress.

---

## YOUR APPS

| App | URL | Purpose |
|-----|-----|---------|
| Planning View | `planning.html` | View crop plans (read-only) |
| Greenhouse | `greenhouse.html` | Track seedlings |
| Field App | `field_app_mobile.html` | Manage field tasks |
| Sowing Sheets | `sowing-sheets.html` | Print crew task lists |
| Bed Assignment | `bed_assignment_COMPLETE.html` | View bed allocations |

---

## DAILY ROUTINE

### Morning (Before Crew Arrives)

1. **Check Sowing Sheets** (`sowing-sheets.html`)
   - Print today's task lists
   - Review what needs to be done
   - Note any special instructions

2. **Check Greenhouse** (`greenhouse.html`)
   - Any trays ready for transplant?
   - Add to today's tasks if needed

3. **Prepare Crew Assignments**
   - Who's doing what
   - Where they'll be working
   - What tools they need

### During the Day

4. **Monitor Progress** (Field App)
   - Check tasks being completed
   - Answer crew questions
   - Reassign as needed

5. **Quality Checks**
   - Walk the fields
   - Check work quality
   - Address issues immediately

### End of Day

6. **Verify Completion**
   - All tasks marked complete?
   - Harvests logged?
   - Any issues to report?

7. **Plan Tomorrow**
   - Check tomorrow's schedule
   - Note any carryover tasks

---

## KEY TASKS

### Viewing the Plan
1. Open `planning.html`
2. Use filters to find specific crops
3. See what's coming up
4. **Note:** You can view but not edit plans

### Checking Greenhouse
1. Open `greenhouse.html`
2. See all active trays
3. Check days since sowing
4. Mark trays ready for transplant

### Printing Task Sheets
1. Open `sowing-sheets.html`
2. Select today's date
3. Click **Print**
4. Distribute to crew

### Handling Issues
- Crew reports problem > Check it yourself
- Serious issue > Call Manager
- Document in scouting reports

---

## WHAT YOU CAN'T DO

These require Manager access:
- Create new plantings
- Edit crop plans
- View sales data
- Access financial info

If you need changes made, ask the Manager.

---

## TIPS FOR SUCCESS

- Print task sheets **before** crew arrives
- Check greenhouse **daily** for transplant-ready seedlings
- Walk fields to verify work quality
- Document everything - harvests, issues, notes
- Communicate changes immediately

---

## NEED HELP?

- Planning questions > Manager
- App problems > Admin
- Equipment issues > Manager
