PWA Icon Generator

Generate all required icon sizes for the Tiny Seed Field App. Click each icon to download, or use "Download All" to get a ZIP.

Instructions

  1. Click "Download All" to download all icon sizes
  2. Place the downloaded PNG files in the /icons/ folder
  3. The icons are named to match the manifest.json requirements
  4. For best results, you can also create a custom icon in a design tool and replace these
