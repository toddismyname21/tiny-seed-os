## SKU NAMING CONVENTION

```
[CROP-3]-[VAR-3]-[UNIT-2]

CROP: 3-letter crop code
VAR: 3-letter variety/type code
UNIT: 2-letter unit code (LB, BN, BG, PT, EA, PK)

Examples:
- TOM-HEIR-LB = Tomatoes, Heirloom, Pound
- BAS-GEN-BN = Basil, Genovese, Bunch
- CSA-MD-20WK = CSA, Medium, 20 Weeks
```

---

## QUICKBOOKS ITEM NAMING

[Category]-[Product]
