## PREPARATION TIMELINE

### 4+ Weeks Before Deadline
- [ ] Review eligibility requirements
- [ ] Gather required documents
- [ ] Update Grant Readiness document
- [ ] Identify matching funds (if required)
- [ ] Request letters of support

### 2 Weeks Before Deadline
- [ ] Draft narrative sections
- [ ] Prepare budget
- [ ] Complete all registration requirements
- [ ] Internal review of application

### 1 Week Before Deadline
- [ ] Final review and edits
- [ ] All attachments compiled
- [ ] Test submission portal
- [ ] Backup plan for technical issues

### Day of Deadline
- [ ] Submit early (not last minute!)
- [ ] Save confirmation/receipt
- [ ] Document submission in tracking

## NOTES

**Grant Application Best Practices:**
1. Never wait until deadline day to submit
2. Build in buffer for technical difficulties
3. Have someone else proofread applications
4. Save all confirmation numbers/receipts
5. Start gathering documents early
6. Maintain relationships with program officers

**Key Contacts for Deadline Questions:**
- VAPG: Check rd.usda.gov
- PA Programs: 717-787-3405 (PA Dept of Agriculture)
- NRCS Programs: Local NRCS service center
- FSA Programs: Local FSA office
