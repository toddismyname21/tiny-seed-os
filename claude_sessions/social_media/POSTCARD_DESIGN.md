## CANVA IMPLEMENTATION GUIDE

### Step 1: Create New Design
- Dimensions: 6.5" x 9"
- Background: #FFF7ED

### Step 2: Front Side
1. Add photo (drag to fill or add with border)
2. Add gold circle (10px, #FFB81C)
3. Add kicker text (Oswald, uppercase, 13pt, dark)
4. Add headline "WON'T YOU BE MY NEIGHBOR?" (Oswald Bold, large)
5. Add gold accent line
6. Add "TINY SEED FARM" logo/text

### Step 3: Back Side
1. Create white rectangle card for offers (rounded corners, shadow)
2. Add 3 offer boxes with gold backgrounds
3. Add body copy (Inter, centered)
4. Create dark CTA box with URL and promo code
5. Generate QR code (use free QR generator with tracking URL)
6. Add contact info
7. Add EDDM indicia placeholder (print shop will add actual indicia)

### Step 4: Export
- Export as PDF (Print quality)
- Include bleed if required by printer
