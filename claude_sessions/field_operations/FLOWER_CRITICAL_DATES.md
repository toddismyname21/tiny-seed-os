## ANNUAL CALENDAR OVERVIEW

```
JAN: Plan/order, start lisianthus, force tulips
FEB: Dahlia splitting, ranunculus pre-sprout, early snaps
MAR: Start warm-season flowers, pot dahlias
APR: Transplant cool-season, succession sunflowers begin
MAY: Transplant warm-season after frost
JUN: Peak production, succession planting
JUL: Fall flower seeding begins
AUG: Overwinter crop seeding, dahlia peak
SEP: Plant tulip bulbs, transplant overwinters
OCT: Direct seed sweet peas, dig dahlias after frost
NOV: Protect overwinters, store dahlias
DEC: Plan next season, early tulip forcing
```

## MONTHLY TASK CALENDAR

### JANUARY
- [ ] Order seeds and supplies
- [ ] Start lisianthus indoors (Dec-Jan)
- [ ] Begin forcing tulips (for March blooms)
- [ ] Plan crop rotation

### FEBRUARY
- [ ] Split dahlia tubers (late Feb when eyes visible)
- [ ] Pre-sprout ranunculus and anemone corms
- [ ] Continue lisianthus care
- [ ] Start eucalyptus, craspedia

### MARCH
- [ ] Pot up dahlia divisions
- [ ] Plant ranunculus/anemones under protection
- [ ] Start celosia, gomphrena, strawflower
- [ ] Ventilate overwinter tunnels

### APRIL
- [ ] Start zinnias, sunflowers, cosmos
- [ ] Remove row covers from overwinters
- [ ] Install trellising for sweet peas
- [ ] First sunflower direct seeding

### MAY
- [ ] Transplant ALL warm season flowers after last frost (May 15+)
- [ ] Direct seed zinnias, sunflowers
- [ ] Plant dahlia tubers in field
- [ ] Harvest overwinters (snaps, stock, sweet peas)

### JUNE
- [ ] Continue succession planting
- [ ] Harvest lisianthus, ranunculus
- [ ] Scout for pests
- [ ] Begin dahlia pinching

### JULY
- [ ] Seed fall snapdragons, stock (Jul 15-Aug 1)
- [ ] Continue successions
- [ ] Harvest zinnias, sunflowers, cosmos
- [ ] Dahlia harvest begins

### AUGUST
- [ ] Seed overwinter dianthus, scabiosa
- [ ] Peak dahlia season
- [ ] Continue successions
- [ ] Order tulip bulbs

### SEPTEMBER
- [ ] Transplant overwinter crops
- [ ] Direct seed larkspur, bells of Ireland
- [ ] Plant tulip bulbs for forcing
- [ ] Order ranunculus/anemone corms

### OCTOBER
- [ ] Direct seed sweet peas (Oct 1-15)
- [ ] Apply row covers after hard frost
- [ ] Dig dahlias after first killing frost
- [ ] Last zinnia/sunflower harvest

### NOVEMBER
- [ ] Clean and store dahlia tubers
- [ ] Check overwinter protection
- [ ] Clean up beds
- [ ] Store equipment

### DECEMBER
- [ ] Start lisianthus (Dec-Jan)
- [ ] Plan next season
- [ ] Order seeds
- [ ] Force tulips for Valentine's Day

## WEATHER TRIGGERS

| Event | Action |
|-------|--------|
| **First frost warning** | Harvest all tender flowers, protect overwinters |
| **Hard freeze (<28°F)** | Cover all overwintered crops |
| **Soil temp 50°F+** | Safe to transplant warm-season flowers |
| **Soil temp 60°F+** | Direct seed zinnias, sunflowers |
| **Last frost passed** | Transplant dahlias, tender annuals |
| **Day length <12 hours** | Stop feeding dahlias |
